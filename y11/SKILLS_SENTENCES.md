# Sentence Generation Restrictions

This log determines EXACTLY what is "unlocked" at each stage in the curriculum. Sentences constructed in any Set MUST NOT violate these unlocks.

## GLOBALLY ALLOWED
- `non` (not), `et` (and), `sed` (but) — these are implicitly unlocked early.
- Proper nouns can be used as needed (e.g. `Roma`, `Marcus`, `Flavia`, `Britanni`) assuming they are obvious to a total beginner.

## NODE 1: 1st Conjugation

**Unlocked Grammar:**
- 1st/2nd Declension Nouns (Nom, Acc, Dat, Abl cases)
- 1st/2nd Declension Adjectives
- Basic Prepositions (`in`, `ad`, `per`, `e/ex`, `a/ab`)
- 1st Conjugation Verbs

**Unlocked Vocab (Set 5):**
**Nouns:** puella, nauta, pecunia, silva, aqua, agricola, femina, villa, servus, amicus, dominus, filius, puer, vir, ager, liber, murus, equus, gladius, nuntius.
**Adjectives:** magnus, parvus, bonus, malus, laetus, iratus, perterritus.
**Prepositions:** ad (acc), in (acc), in (abl), per (acc), e/ex (abl), a/ab (abl).

**Set 8 (Present Tense Sentences):**
- Limits: Present Tense *only*. Nom and Acc *only*. No prepositions yet (to ease them in).

**Set 12 (Present System - Pres/Imp/Fut/Inf):**
- Limits: Present, Imperfect, Future, Infinitives.
- Unlocks: `in + abl` and `in + acc` and `ad`.

**Set 16 (All Active Tenses):**
- Limits: Present, Imperfect, Future, Perfect, Pluperfect.
- Unlocks: Dative case (`do`, `nuntio`), Ablative (`e/ex`, `a/ab`, `per`).

**Set 18 (PPP Sentences):**
- Limits: Include PPPs modifying nouns in Nom/Acc. *Advanced preview:* `necatus est` (perfect passive) is allowed as a special preview.

## NODE 2: 2nd Conjugation

**Unlocked Grammar:**
- 2nd Conjugation Verbs (video, habeo, teneo, iubeo, appareo, persuadeo)
- Accusative + Infinitive (e.g., `iubeo milites oppugnare`)

**Unlocked Vocab (Set 21):**
**Nouns:** bellum, periculum, castra(pl), consilium, donum, praemium, imperium, auxilium.
**Adverbs:** subito, statim, mox, diu, quoque.

**Set 24 (Present Tense):**
- Limits: Nom/Acc/Dat/Abl. Present tense. Must include 2nd conjugation prominently alongside 1st conjugation review.
